# monster_print

This Prooject is a website for Monster Print Store.

## Author

- Name: Abdallah Miri

## Description

This project is a website for Monster Print Store, which offers various printing services and products. The website provides information about the services offered, pricing, and contact details for customers to reach out for inquiries or orders.

## Technologies Used

- HTML
- CSS
- JavaScript
- Tailwind CSS
- GitHub Pages
- Jykyll

## Features

- Responsive design for optimal viewing on various devices.
- Clear navigation to different sections of the website.
- Contact information for customer inquiries.
- Information about printing services and pricing.
